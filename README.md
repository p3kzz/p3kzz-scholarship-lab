# 🎓 Scholarship Lab

Scholarship Lab adalah platform berbasis AI yang membantu pelajar menemukan dan mencocokkan beasiswa berdasarkan profil dan kualifikasi mereka.

## 📦 Project Structure

- frontend → React + Vite
- backend → Express API
- ai-service → FastAPI (AI)
- docs → dokumentasi & proposal
